# Privacy Manifesto

Privacy Manifesto est un blog statique (Jekyll) centré sur la vie privée, les droits numériques et la critique des plateformes.

## Objectif

- Publier des manifestes documentés sur les enjeux de surveillance et d'exploitation des données.
- Proposer une base éditoriale claire, publique et versionnée.

## Stack

- Jekyll
- Theme Leonids (`renyuanz/leonids`)
- GitHub Pages (deploiement via GitHub Actions)
- Jekyll Polyglot (FR/EN)

## Structure

- `_manifestos/` : manifestes
- `_posts/` : billets de blog
- `_config.yml` : configuration principale
- `.github/workflows/pages.yml` : pipeline de deploiement
- `doc/README.md` : documentation technique et d'exploitation
